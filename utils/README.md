# utils 工具部分函数说明

## split_word_meaning(meaning)

### 所属文件
utils/split_meaning.py

### 功能
拆分单词释义字符串，提取词性和对应翻译。

### 输入规则
- 词性和中文翻译用**一个空格**隔开；
- 单种词性内，同一种意思的不同描述用**英文逗号**隔开；
- 单种词性内，不同意思用**英文分号**隔开；
- 多种词性之间用**两个空格**隔开；
- 标准词性识别规则：**长度≤6**且以**英文句点**结尾（如 n.）。

### 所有标准词性参考
| 词性     | 含义    |
|--------|-------|
| n.     | 名词    |
| v.     | 动词    |
| vi.    | 不及物动词 |
| vt.    | 及物动词  |
| adj.   | 形容词   |
| adv.   | 副词    |
| prep.  | 介词    |
| conj.  | 连词    |
| pron.  | 代词    |
| num.   | 数词    |
| art.   | 冠词    |
| int.   | 感叹词   |
| aux.   | 助动词   |
| modal. | 情态动词  |

### 输入示例
```python
meaning = "n. 著作家,作者;创始人  v. 写作"
split_word_meaning(meaning)
```

### 输出示例
```python
[
    {"pos": "n.", "trans_list": ["著作家,作者", "创始人"]},
    {"pos": "v.", "trans_list": ["写作"]}
]
```
