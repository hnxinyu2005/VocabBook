# VocabBook

### words.csv 表头释义

| 表头字段            | 数据类型 | 字段含义                                         | 示例值    | 是否可空  |
|-----------------|------|----------------------------------------------|--------|-------|
| `word`          | 字符串  | 核心单词（唯一值，无重复）                                | work   | False |
| `phonetic`      | 字符串  | 单词音标（建议用英式音标，格式统一为/.../）                     | /wɜːk/ | True  |
| `pos_count`     | 整数   | 该单词的词性数量（用于快速遍历词性 / 释义，避免空值判断）               |.n| False |
| `pos1`          | 字符串  | 	第 1 个词性（如 n. 名词 /v. 动词 /adj. 形容词 /adv. 副词等） |        | False |
| `meaning1`      | 字符串  | 第 1 个词性对应的释义                                 |        | False |
| `pos2`          | 字符串  | 第 2 个词性（无则留空）                                |        | True  |
| `meaning2`      | 字符串  | 第 2 个词性对应的释义（无则留空）                           |        | True  |
| `pos3`          | 字符串  | 	第 3 个词性（无则留空，最多支持 3 个词性，满足绝大多数单词需求）         |        | True  |
| `meaning3`      | 字符串  | 第 3 个词性对应的释义（无则留空）                           |        | True  |
| `example`       | 字符串  | 单词例句原文                                       |        | True  |
| `example_trans` | 字符串  | 例句中文翻译                                       |        | True  |
| `textbook`      | 字符串  | 该单词所属的资料书名称                                  |        | True  |
| `unit`          | 字符串  | 该单词所属资料书的单元号和页码信息等                           |        | True  |
| `review_count`  | 整数   | 该单词被考察的总次数                                   |        | True  |
| `correct_count` | 整数   | 该单词被答对的次数                                    |        | True  |
| `last_review`   | 字符串  | 最后一次考察时间（格式：YYYY-MM-DD HH:MM:SS）             |        | True  |
