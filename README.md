# VocabBook
VocabBook 是一款基于 Python 开发的轻量级桌面单词记忆工具，主打极简、高效、完全自定义的单词学习体验。  
专为习惯使用纸质书籍背单词，但难以自测记忆效果、无法系统化复习的用户设计，让纸质学习与电子化自测、巩固无缝结合。
- 支持全自定义单词库：可按纸质书章节、学习主题等维度自主录入、管理单词，精准匹配个人学习进度；
- 主打高效轻量化体验：本地运行无广告，聚焦单词自测、错题复盘等核心功能，快速巩固纸质书记忆的单词，弥补纸质记词 “复习不便、无即时反馈” 的不足；
- 交互极简，无需复杂操作，让用户专注于单词记忆本身，是纸质背词场景下的便捷辅助工具。

项目 GitHub 地址： https://github.com/hnxinyu2005/VocabBook

## 使用方法
使用说明：软件导入单词时需要严格的格式文本，纯手动整理、排版会比较困难，推荐使用生成式 AI 大模型，快速生成符合要求的单词导入文本。

### 部分使用方法

#### 导入单词
导入单词时推荐使用生成式 AI 大模型将单词书图片转换为符合要求的单词导入文本。

第一步：使用手机相机将单词书的一页或者多页拍下来。

第二步：将照片导入 AI 大模型（如“豆包”），并且附上以下 Prompt。
```text
我需要将照片中的单词转换为特定的文本以导入记单词的软件，你按照以下要求输出我需要的文本。
```

### words.csv 表头释义

| 表头字段            | 数据类型 | 字段含义                                                                                                         | 示例值                                                                                                 | 是否可空  |
|-----------------|------|--------------------------------------------------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------|-------|
| `word`          | 字符串  | 核心单词（唯一值，无重复）                                                                                                | author                                                                                              | False |
| `phonetic`      | 字符串  | 单词音标（建议用英式音标，格式统一为/.../）                                                                                     | /ˈɔːθə(r)/                                                                                          | True  |
| `meaning`       | 字符串  | 词性以及对应的意思（词性与中文翻译之间用一个空格隔开；如单种词性的同种意思的不同描述，用英文逗号隔开；如单种词性有不同意思，用英文分号隔开；如有多种词性，两种词性之间用两个空格隔开；能够自动识别的标准词性在下方列出） | n. 著作家,作者;创始人,发起者                                                                                   | False |
| `example`       | 字符串  | 单词例句原文（如有多个，用分号隔开；句中出现的单词可选用“**”包裹以凸显）                                                                       | Our guest on today's "Book Talk" is John Black, the **author** of the new bestseller, Retire Early. | True  |
| `example_trans` | 字符串  | 例句中文翻译（如有多个，用分号隔开，需要与 `example` 的顺序对应）                                                                       | 我们今天的《图书访谈》栏目嘉宾是约翰·布莱克，他是畅销新书《提前退休》的作者。                                                             | True  |
| `textbook`      | 字符串  | 该单词所属的资料书名称或者网站                                                                                              | 新东方《四级词汇》乱序版                                                                                        | True  |
| `unit`          | 字符串  | 该单词所属资料书的单元号和页码信息或者网址等                                                                                       | Word List 1; p1                                                                                     | True  |
| `review_count`  | 整数   | 该单词被考察的总次数                                                                                                   |                                                                                                     | False |
| `correct_count` | 整数   | 该单词被答对的次数                                                                                                    |                                                                                                     | False |
| `last_review`   | 字符串  | 最后一次考察时间（格式：YYYY-MM-DD HH:MM:SS）                                                                             |                                                                                                     | False |
> 注：`review_count`、`correct_count` 默认值为 0， `last_review` 没有记录时为空字符串。以上三项由系统生成，无需填写。
